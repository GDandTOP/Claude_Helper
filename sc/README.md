# SuperClaude 명령어 (`sc`)

이 폴더는 **SuperClaude** 패키지에 포함된 **슬래시 명령어** 설명입니다.  
사용자가 터미널에서 `superclaude install`을 실행하면, 이 명령어들이 `~/.claude/commands/sc/` 경로에 설치됩니다.

> **슬래시 명령어란?**  
> Claude Code 등에서 `/`로 시작하는 짧은 명령으로, 특정 작업(에이전트 호출, 검색 등)을 빠르게 실행할 때 씁니다.

---

## 사용 가능한 명령어

| 파일 | 설명 |
|------|------|
| **agent.md** | 목적에 맞는 **전문 AI 에이전트**를 다루는 명령 |
| **index-repo.md** | 저장소를 **인덱싱**해서 컨텍스트(맥락)를 효율적으로 쓰기 위한 명령 |
| **recommend.md** | 상황에 맞는 **명령어 추천** |
| **research.md** | **웹 검색을 병렬로** 돌려 깊게 조사하는 연구용 명령 |
| **sc.md** | SuperClaude에서 쓸 수 있는 **명령 목록 전체** 보기 |

---

## 수정·배포 시 주의사항

이 `sc` 아래 명령어들은 패키지 배포용으로 **다른 경로의 원본을 복사한 것**일 수 있습니다.

명령어 내용을 바꿀 때는 보통 아래 순서를 지킵니다.

1. **`plugins/superclaude/commands/`** 안의 파일을 먼저 수정합니다.  
2. 같은 내용을 **`src/superclaude/commands/`** 에도 반영합니다.  
3. **두 위치가 항상 같아야** 합니다 (한쪽만 고치면 배포·동기화가 어긋날 수 있음).

**v5.0**부터는 플러그인 시스템이 `plugins/` 폴더를 **직접** 사용하는 방향으로 바뀔 예정입니다.
